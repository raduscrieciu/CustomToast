# CustomToast

Step 1. Add the JitPack repository to your build file
```text
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

Step 2. Add the dependency
```text
	dependencies {
	        implementation 'com.github.raduscrieciu:CustomToast:Tag'
	}
```
